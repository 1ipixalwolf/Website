Quiz: Accessible Tables



When a PDF is not formatted correctly, screen readers will just say "image" when you try to read the document. Improperly formatted PDFs are the mor common type. Consider using another document type that tends to be more accessible. This does not Entail learning some obscure new document type. Standard ones such as .doc, .dox, and .txt generally work fine.
