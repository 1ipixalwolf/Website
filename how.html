Take a few extra moments to create thoughtful labels and structure. Make sure your website relies on semantic HTML elements rather than just graphical CSS. Any graphics should be backed with text. Make sure to have clear formatting and structure. This thoughtful text based approach will not only enhance the experience of screen reader users but will improve things for everyone. The user interface will just be better and clearer. Spread the word. The more people Understand the benefits of clear HTML, the better things will get for usability for everyone.
